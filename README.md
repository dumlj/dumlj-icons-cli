# dumlj/icons

svg 图标自动化交付方案

## packages

### @docs

图标库在线文档

### @dumlj/icons

自动生成的图标库

`yarn add @dumlj/icons`

```
import { Icon } from '@dumlj/icons'
```

### @dumlj/icons-cli

生成图标脚本

`yarn add @dumlj/icons-cli -D`

```
dumlj-icons -c ./config.js
```

### figma-publish-notice

图标库发布通知 figma 插件
